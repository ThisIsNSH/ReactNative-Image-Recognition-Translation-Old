# ReactNative-TIRAT
### Text in Image Recognition and Translation. (In Dev State)

The project is a cross platform app which detects and extracts the text in an image and provides translations in languages selected by the user. 

### Used

* React
* React Native
* ObjectiveC Xcode
* Java AndroidStudio
* JavaScript CSS
